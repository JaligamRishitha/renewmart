Available Routes

Project Details Page

Full-page view of project details with task tracking and completion status.

Navigation Sources

These routes can be accessed from multiple places in the application:

  • • Admin Investor Interests: Click "View Project Details & Tasks" button
  • • Marketplace: Click "View Details" button on project cards
  • • Investor Details Modal: Click "View Project Details & Tasks" button
  • • Direct URL: Navigate to /project-details/:landId

Features

  • ✅ Project Overview: Visual progress indicators and project metadata
  • ✅ Task Tracking: Real-time task completion status and progress bars
  • ✅ Task Details: Individual task cards with assignees and due dates
  • ✅ Responsive Design: Works on desktop and mobile devices
  • ✅ Navigation: Back button and breadcrumb navigation
  • ✅ Error Handling: Graceful handling of missing projects
  • ✅ Loading States: Loading indicators while fetching data

Route Structure

/project-details/:landId

Where :landId is the unique identifier for the project/land.
